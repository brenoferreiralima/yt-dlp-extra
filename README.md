
# YDE (yt-dlp-extra)
a basic shell script to make yt-dlp simpler to use (with a few personal options)

## Dependencies
- coreutils (tee)
- expect (unbuffered)


## Install
```sh
git clone --depth=1 https://github.com/brenoferreiralima/yt-dlp-extra.git
```
```sh
cd yt-dlp-extra
```
```sh
sudo ln -sf yde /usr/local/bin
```


## Uninstall
```sh
sudo rm /usr/local/bin/yde
```


## Usage
```sh
yde url-to-download
```
